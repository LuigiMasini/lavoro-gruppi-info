1 /*===============================================================================================
2  PlayStream Example
3  Copyright (c), Firelight Technologies Pty, Ltd 2004-2016.
4 
5  This example shows how to simply play a stream, such as an mp3 or wav.
6  The stream behaviour is achieved by specifying FMOD_CREATESTREAM in the call to
7  FMOD_System_CreateSound.
8  This makes FMOD decode the file in realtime as it plays, instead of loading it all at once.
9  This uses far less memory, in exchange for a small runtime cpu hit.
10 ===============================================================================================*/
